Restyler

Hello!

Restyler lets you totally restyle the appearance of every webpage with an easy to use interface.
Ongoing changes in the page are automatically observed so new styling is always taken into account.
Perfect for testing out design changes or rebranding on an existing page!

How to use?

  1. Open the Developer Tools window (Ctrl+Shift+I or Ctrl+Opt+I in Mac) and goto the Restyler tab.
    You can try it out on this page!
  2. Pick an attribute you would like to change, for example "color".
    Let's add a style rule that will replace any element with original value of "white" to "green".
    We can also change all color values to "green" if we leave the original value field empty.
  3. Click "set" and watch the result!
  4. You can add more styling rules, all CSS attributes are supported.
    You can also apply the style change to specific elements if you give a selector to match elements upon.
  5. You can also share your changes to other Restyler users or export the style as a CSS file for front end designers.
  6. Finally, check out the text editing feature!
    Turn it on and every text in the page is editable!

Comments:

  • Text editing mode is not saved across sessions (reloading the page will revert the text back to original).
  • Styling rules are applied by the their order, you can rearrange the order by dragging the left side handle.
  • You can use any supported CSS values (for example for "color": rgb, hex, names, etc...).
  • Make sure you write the valid units for CSS values.
    For example: "font-size" should have "px", "rem", etc...
